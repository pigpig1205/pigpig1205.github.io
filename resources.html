Codecademy

-Codecademy is an online platform that provides free coding courses for many different languages, like Python, HTML and JavaScript.


CodeCombat

-CodeCombat is an educational game that teaches you how to code through fun levels and bonus courses. Choose from Python, JavaScript, Lua, Io, CoffeeScript and Clojure or even a mix of them and begin your coding adventure!


Hour of Code

-Hour of Code, made by Code Studio, teaches students languages ranging from Scratch and Blockly to Java and Rust, with short tutorials to watch and simple goals to reach.


CodePen

-CodePen is an online community that you can find, make, test and post HTML, CSS and JavaScript code on.


GitHub

-GitHub is an online hosting service for posting webpages and other programs, such as Python.
